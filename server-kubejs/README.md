# Серверная связка GondurasMC (KubeJS)

Команда `/claim` (выдача донат-инвентаря с сайта) + учёт наигранного времени.
Работает без компиляции мода — через **KubeJS** (уже есть в модпаке).

## Установка на сервер (Ultraservers / Pterodactyl)
1. В `gondurasmc.js` впиши `KEY` = значение `SERVER_KEY`, заданное на сайте
   (его НЕТ в репозитории намеренно — спроси/возьми из секретов Fly).
2. Через File Manager положи `gondurasmc.js` в папку сервера:
   `kubejs/server_scripts/gondurasmc.js`
3. В консоли сервера выполни `/reload` (или перезапусти сервер).
4. Проверка: зайди в игру, напиши `/claim` — выдаст предметы из инвентаря сайта.

## Как работает
- **/claim** → сервер запрашивает `GET /api/claim` у сайта (по ключу), получает
  список предметов, выдаёт их через `/give`, сайт очищает инвентарь.
- **Время игры** → раз в минуту сервер шлёт `POST /api/playtime` по каждому
  онлайн-игроку (+60 сек). Сайт копит время за сегодня и пускает к ежедневному
  кейсу при ≥ 2 ч и не чаще раза в 24 ч.
- HTTP-запросы идут в фоновом потоке (не лагают сервер); выдача предметов — в
  главном потоке (безопасно).

## Если KubeJS заблокирует java.net
Некоторые сборки KubeJS ограничивают сетевые классы. Если в логе будет ошибка
`java.net.URL` — скажи, переведу связку на RCON (выдача с сайта) или на маленький
NeoForge-мод.
